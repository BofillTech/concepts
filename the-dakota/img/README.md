# The Dakota — Image Manifest

This concept hotlinks real property photos directly from the client's own
live WordPress uploads (`staydakota.com/wp-content/uploads/...`) with
`referrerpolicy="no-referrer"`. They render in a real browser / on GitHub
Pages (served from the client's own active site). For a bulletproof WP build,
re-host each via `wp_upload_media` (server-side fetch) and drop into `/img/`.
