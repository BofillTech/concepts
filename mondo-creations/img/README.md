# Mondo Creations — Image Assets

All photos and the logo are **real Mondo Creations assets**, re-hosted from the client's
WordPress CDN (`mondocreations.com/wp-content/uploads/`) to bofilltech.com via `wp_upload_media`
(server-side fetch) so the concept is bulletproof on GitHub Pages and won't break if the client
edits their own site. For the WordPress theme conversion, download these into `/img/` and update
the paths in `index.html` + `css/style.css`.
